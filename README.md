## Employee Portal - MVC Project
A comprehensive Employee Data system designed to demonstrate the fundamentals of the ASP.NET Core MVC architectural pattern. 
This project provides a user-friendly interface for managing employee data, including professional details, departmental information, and CRUD operations.

### 🛠️ Features
**CRUD Operations:** Full Create, Read, Update, and Delete functionality for employee records.
**MVC Architecture:** Clear separation of concerns between Models, Views, and Controllers.
**Database Management:** Entity Framework Core used for seamless database interactions.
**Data Validation:** Secure form submission with server-side validation.
**Secure Login:** Authorization and Authentication for access to the pages while managing the state troughout the project scope.

### 🚀 Technologies 
**Used Framework:** ASP.NET Core MVC (.NET 8)
**Database:** SQL Server / LocalDB
**ORM:** Entity Framework Core
**Frontend:** Razor Views, Bootstrap 5, Javascript

### 🏗️ Architecture Components
**Model:** Defines the Employee entity structure and database mapping (via EF Core).
**View:** Razor templates for displaying data tables, edit forms, and create interfaces.
**Controller:** Handles incoming requests, interacts with the model, and renders views.
